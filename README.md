Kyrgyz MNIST Classification with CNN

Описание проекта

Этот проект посвящён классификации рукописных символов киргизского MNIST (цифры и буквы). Основная цель — построить высокоэффективную модель для распознавания символов и подготовки предсказаний для сабмита на Kaggle.

Используется Convolutional Neural Network (CNN) с несколькими сверточными слоями, Batch Normalization, Dropout и полносвязными слоями для классификации 36 классов (символы 1–36)

Файл:
Kyrgyz_MNIST_CNN.ipynb — Jupyter/Colab ноутбук с кодом модели и визуализацией

Используемые технологии:
Python 3.x
PyTorch
Pandas, NumPy
Matplotlib/Seaborn (для визуализации)
GPU ускорение через CUDA (опционально)

Архитектура модели:
Модель представляет собой CNN с улучшениями:
Сверточные слои с ReLU и BatchNorm для извлечения признаков из изображений
MaxPooling для уменьшения размерности и устойчивости к смещениям
Dropout в полносвязных слоях для регуляризации
Полносвязные слои для классификации на 36 классов

Особенности:
Использована нормализация пикселей (деление на 255.0)
Модель обучена с кросс-энтропийной функцией потерь (CrossEntropyLoss)
Оптимизатор: Adam
Обучение на GPU (Т4)
